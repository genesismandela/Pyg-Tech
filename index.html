Pig Game Description

  • The game has two players who are playing by rounds.
  • In each turn, a player rolls a dice as many times as wishes and the result get added to the round socre.
  • When rolling the dice if result is 1, the round score get lost and the next player plays.
  • If player choose to hold the current score, it is added to the personal global score and the next player plays.
  • The first player to reach 100 points on global score wins the game.

Pig Game Structure

First of all to start creating the game it is necessary to make the html. For that we create two div elements that contains each player information. In each of them we can see the players name and below, the global score that is updated every time that a player holds the current score, and as we know, is the sum of every result when dice is rolled less when result is one.
To make the game work, we just have to include three button elements. One is roll-dice, the other is hold-score and the last one is an option to reset all to start a new game.

Pig Game Style

The css style of the game is pretty simple, every player has his own panel where the information already explained. We included the basics for font-style and some position elements to align and transform the text, as well as some margin and paddings.
Need to say that everytime that a player is playing the class active is added to the panel of current player, where the style changes to know that this is the one who's playing. We will explain how it changes in JavaScript section.
The page does have few more elements to style the buttons and the dice image, all of them in the middle of the page and the buttons has some hover pseudo element to make it more accesible when playing the game.

Pig Game Functionality

New Game

We start creating the JavaScript file declaring four variables but not giving them any value. We are giving them value inside a function that we create to make the game start from 0.
In said function we specify that when the game starts the dice image will be hidden. document.querySelector('.dice').style.display = 'none';. We make global and current scores start from 0 but this time using document.getElementById('score-0').textContent = '0'; and document.getElementById('current-0').textContent = '0';, one for each player.
The game starts with the name of each player on the top of each panel, which is changed when game is over.

Roll Dice

To start playing the game, we create an event to make the roll dice and get numbers randomly. For that instead of using an element to change style, we use document.querySelector('.btn-roll').addEventListener('click', function(), specifiying that when button is clicked something will happen.
First of all we make a random number appear using dice = Math.floor(Math.random() * 6) + 1;. When this is executed we make the dice appear again and include the number we got in the scr of the element created for the dice image in the html diceDOM.src = 'dice-' + dice + '.png';.
The last thing we want when we roll the dice is to reset the round score to 0 and change the player if the given number is equal to 1.

Next Player

To make the next player play we create a function and for first time we use a ternary operator activePlayer === 0 ? activePlayer = 1 :. Here we say that if the player who is playing is 0, will change to 1 and vice versa.
For first time we use the toggle element to make a class selector change from one player to another document.querySelector('.player-0-panel').classList.toggle('active');.
When player changes we want the current score to reset to 0 and hide the dice for the new player.

Hold Score

When a player is playing his turn and wants to hold the current score we create an event function to keep it in global score scores[activePlayer] += roundScore;.
As we know from the game rules if any player reaches 100 points wins the game. To make this we use a boolean expresion.

Game Over

When a player reaches a score of 100 the game is over and all functionalities for the roll and hold button are disabled. To start a new game in the already created function for new game, we included some functionalities to reset the parameters from the last game using two new functions remove and add.

document.querySelector('.player-0-panel').classList.remove('winner');, one for each player.
document.querySelector('.player-0-panel').classList.remove('active');, one for each player.
document.querySelector('.player-0-panel').classList.add('active');.
